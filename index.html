FTNd

Fidonet/FTN Services for Unix

Tools:
Navigation: Home | FTNd | mbsebbs | Net-Citadel | pkg-sbbs | sbbsdev

FTNd is an experimental system for Fidonet/FTN services that runs on Linux and other POSIX compatible operating systems and is not limited to PC's. The package including a mailer, tosser, ticfile processor, filefind and other utilities. The mailer supports FTS-0001, YooHoo/2U2, EMSI protocols over modem, TCP/IP IFC and Binkp protocols. Zedzap, Zmodem, Telink and Hydra file transfer protocols. Full FTN mail support, including automatic routing for hub and host systems. Internal mail format is JAM (c) messagebase. Full tic file support, including extended tic files.

(See the MBSE BBS Debian Package home page for more details regarding the application this one was forked from.)

  • The debian branch in the repository is used for the development of the debian packaging directory for the stable versions of FTNd.
  • The debian-dev branch in the repository is used for the debian packaging directory for development versions of FTNd.
  • The ftnd-dev branch in the repository is used for the development version of FTNd.
  • The master branch in the repository contains the current code for the FTNd application itself..

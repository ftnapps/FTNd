MBSE BBS

Debian packaging for MBSE BBS

Tools:
Navigation: Home | mbsebbs | Net-Citadel | pkg-sbbs | sbbsdev
  • This project is primarily about Debian packaging for MBSE BBS, which is a BBS system that runs on Linux and other POSIX compatible operating systems and is not limited to PC's. See the MBSE BBS home page for more details regarding the application itself.
  • The debian branch in the repository is used for the development of the debian packaging directory for the stable versions of MBSE BBS.

The master branch in the repository contains the code for the MBSE BBS application itself, including that imported from the authors Mercurial (HG) Repository.

ftnpasswd - The password wrapper.

Description.

ftnpasswd is the wrapper for the passwd program is present on all GNU/Linux systems. To use passwd to change the password of another user is only allowed by root. The ftnpasswd program overcomes this limitation. The wrapper ftnpasswd is run from the bbs by user ftnd when a new user is being registered to set his password. It can also be started from the bbs when an existing user changes his password. If you as sysop use ftnsetup to change a users password it will be used too. His password under Unix is then always the same as his password in the bbs program. This is necessary for the user to be able to get his email using the pop3 protocol.

You never need to run ftnpasswd by hand, in fact it is protected so that it can only be started by the bbs or ftnsetup.

 

Environment.

ftnpasswd must be installed setuid root and setgid root, ls -la looks like this:

-rws--s--x   1 root     root         6644 Jun 26 21:23 /opt/ftnd/bin/ftnpasswd*

 

Commands.

Not mentioned here because ftnpasswd is only called by other programs, running manually is not supported.

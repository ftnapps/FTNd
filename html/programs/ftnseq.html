ftnseq - Sequence number creator

Synopsis.

ftnseq

 

Description.

ftnseq writes a eight character hexadecimal unique sequence number to the stdout. This number is received from ftntask which keeps track of the generated sequence numbers. This written number can be used in shell scripts to create unique filenames for Fidonet .pkt files, for example:

cp temp.pkt `ftnseq`.pkt

 

Bugs.

Nah, it's only 50 lines code, what could go wrong?

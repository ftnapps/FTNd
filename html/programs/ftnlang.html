ftnlang - Language Data Compiler

Synopsis.

ftnlang [language data file] [language source text]

 

Description.

ftnlang compiles the source textfile to language datafile which is used by the ftnbbs program. You only need to use this program if you install a new language file. When you build the complete FTDd package, this command is run automatic for you.

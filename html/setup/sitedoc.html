FTNd Setup - Create Sitedocs.

Create Sitedocs

This option creates everal documents in the share/doc and share/doc/html directories under the home directory of FTNd. In the first directory is a file called site.doc that is formatted to print. In the html directory are a lot of html pages with cross reference links. Since these pages contain confidentional information of your users and links, you should not make these html pages public available, but you can browse these with a browser with open file and point to /opt/mbse/share/html/index.html.
